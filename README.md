# 💰 FinSight – Personal Finance Manager

A modern and responsive Personal Finance Manager built with **React.js**, **Vite**, **Tailwind CSS**, and **Recharts**. FinSight helps users securely manage their income, expenses, budgets, and financial insights through an intuitive dashboard and analytics.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/rishabhpandey-2005/FinSight-Personal-Finance-Manager.git
```

Go to project folder

```bash
cd FinSight-Personal-Finance-Manager
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```

Build production version

```bash
npm run build
```
